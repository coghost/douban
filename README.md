# douban

#### douban fm on ubuntu
***


#### usage:
```
Usage: aas [OPTION]... [SCRIPT]...

	h/help
                show all usages.
	c/channel
                   get channel.
	cl/clean
                clean mplayer process.
	sc/setchannel
                   set channel.
	l/playlist
                show current channel and playlist.
	p/n/play/nextchannel_id(optional)
                running play songs.
	s/stop
                  stop playing.
	q/quit
                         quit .


```
